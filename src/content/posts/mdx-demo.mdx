# MDX 博客系统演示

欢迎体验全新的 MDX 博客系统！这里可以将 React 组件无缝嵌入到 Markdown 内容中。

## 什么是 MDX？

MDX 是 Markdown 和 JSX 的强大组合，让我们能够：

- 📝 **编写 Markdown**：保持熟悉的写作体验
- ⚛️ **嵌入 React 组件**：添加交互式内容
- 🎨 **自定义样式**：完全控制页面外观
- 🔧 **扩展功能**：轻松添加新特性

## 代码示例

这是一个标准的代码块：

```javascript
function HelloWorld() {
  return <div>Hello, World!</div>
}

// 现在这个组件可以直接在 MDX 中使用！
```

## 交互式内容

未来我们可以在这里添加各种 React 组件：

- 🔥 **评论系统**：让读者参与讨论
- 📊 **图表组件**：数据可视化
- 🎮 **交互演示**：在线代码编辑器
- 📧 **订阅表单**：收集用户邮箱

## 数学公式支持

MDX 还可以轻松集成数学公式渲染：

```
E = mc²
```

## 自定义组件示例

看，这是一个真正的 React 组件，直接嵌入在 Markdown 中：

<InteractiveCounter initialValue={5} title="MDX 演示计数器" />

你可以点击按钮与组件交互！这展示了 MDX 的强大之处：

```jsx
// 在 MDX 中使用 React 组件就是这么简单：
<InteractiveCounter initialValue={5} title="MDX 演示计数器" />
```
